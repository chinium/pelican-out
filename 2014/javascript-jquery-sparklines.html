Javascript - jQuery Sparklines

Thu 28 August 2014 Category: IT Tag: Javascript, Chart

sparkline이란 일반적으로 축(axes)이나 좌표(coordinates) 없이 그려진 작은 라인 차트를 의미한다.

온도나 주가와 같이 시간에 따라 변화하는 형태를 간단하면서 함축적으로 나타낼 수 있어
문장 중간이나 여러 차트를 그룹지어서 표시할 때 유용하다.
