SQL – 최근 30일 날짜 리스트

Wed 11 April 2018 Category: IT Tag: SQL

이번에는 '최근 30일 날짜 리스트'를 출력하는 SQL Query를 살펴보자.

Query (MySQL)

SELECT DATE_SUB(CURDATE(), INTERVAL offset DAY) AS DT
FROM
(
  SELECT C.days AS offset
  FROM
  (
    SELECT a*10+b AS days
    FROM
    (
      SELECT 0 a
      UNION ALL SELECT 1
      UNION ALL SELECT 2
      UNION ALL SELECT 3
    ) A,
    (
      SELECT 0 b
      UNION ALL SELECT 1
      UNION ALL SELECT 2
      UNION ALL SELECT 3
      UNION ALL SELECT 4
      UNION ALL SELECT 5
      UNION ALL SELECT 6
      UNION ALL SELECT 7
      UNION ALL SELECT 8
      UNION ALL SELECT 9
    ) B
  ) C
  WHERE C.days BETWEEN 0 AND 29
) D
ORDER BY DT
